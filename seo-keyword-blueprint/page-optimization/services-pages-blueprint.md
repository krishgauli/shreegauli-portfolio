# Services Pages Blueprint

## Page Goal

Use the services section to route visitors cleanly into the right commercial page and give each core service one clear keyword owner.

## Services Overview Page

- `URL`: `/services`
- `Primary keyword`: `digital marketing services`
- `Secondary`: `seo services`, `google ads consultant`, `social media consultant`, `marketing automation services`
- `H1`: `Digital Marketing Services Built Around Bottlenecks`

### Recommended Sections

- Service selector by problem
- Four core service summaries
- When to choose each service
- Proof snapshots
- FAQ
- CTA to book

## Core Service Page Map

| URL | Primary keyword | Secondary keywords | Main CTA |
|---|---|---|---|
| `/services/seo` | `seo consultant` | `seo services`, `technical seo audit`, `seo audit service`, `keyword research services` | Book an SEO strategy call |
| `/services/paid-media` | `google ads consultant` | `google ads management services`, `google ads audit`, `ppc consultant`, `meta ads consultant` | Request an ad account review |
| `/services/social-media` | `social media consultant` | `social media management services`, `linkedin marketing strategy`, `instagram growth strategy` | Start a social strategy conversation |
| `/services/automation` | `marketing automation consultant` | `marketing automation services`, `n8n automation`, `crm automation consultant` | Book an automation scoping call |

## Shared Service Page Template

1. Outcome-led hero
2. Who this service is for
3. Problems it solves
4. Process and deliverables
5. Metrics or outcomes to expect
6. Related case study
7. FAQ
8. CTA

## H2 Structure Template

- `H2`: Who this is for
- `H2`: The bottlenecks this solves
- `H2`: What the engagement includes
- `H2`: How the work is executed
- `H2`: Related results and proof
- `H2`: Frequently asked questions

## Internal Links To Add On Every Service Page

- Parent services page
- One matching case study
- Two matching blogs
- `/book`
- `/contact`

## Trust Elements To Include

- Specific tools used
- Clear deliverables
- Expected timelines
- Quantified proof from a case study
- FAQ answers grounded in real objections

## Conversion Elements To Include

- Sticky or repeated CTA
- Contact form at the bottom
- Optional mini-offer like audit or review for SEO and paid media

## SEO Notes

- Each page needs one clear primary keyword owner
- Keep major synonyms in title, H1, intro, and one H2, but do not repeat them unnaturally
- Use FAQ schema on all four core service pages

## Common Mistakes To Avoid

- Reusing the same copy structure without service-specific proof
- Targeting the same keyword on the overview page and the detail page
- Sending traffic back to the homepage instead of a service CTA
